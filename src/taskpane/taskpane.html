How to Use

  1. Use the PROMPT function: =PROMPT(message, model, apiKey, systemPrompt, provider)
  2. For streaming responses, use PROMPT_STREAM: =PROMPT_STREAM(message, model, apiKey, systemPrompt, provider)

Parameters:

  • message: The prompt message to send to the AI.
  • model: The AI model to use. See provider documentation for available models:
    • OpenAI Models
    • Anthropic Models
  • apiKey: Your API key for the chosen provider. Can be a direct string or a cell reference.
  • systemPrompt: (Optional) A system prompt to provide context for the AI.
  • provider: The AI provider to use ("openai" or "anthropic").

Note: Both OpenAI and Anthropic integrations are available. Ensure you use the correct API key for the chosen provider.
